# ⚙️ EcoLink - Backend Server

السيرفر المركزي والـ APIs الخاصة بمنصة **EcoLink**، مبني باستخدام بيئة Node.js وإطار عمل Express وقاعدة بيانات MongoDB.

## 🛠️ المكتبات والتقنيات المستخدمة (Dependencies)

* **`express`**: إطار العمل الأساسي لبناء السيرفر والـ RESTful APIs.
* **`mongoose`**: الـ ODM للاتصال بقاعدة بيانات MongoDB وإدارة الـ Schemas والعلاقات.
* **`dotenv`**: لقراءة المتغيرات السرية والبيئية من ملف `.env` (مثل روابط قاعدة البيانات ومفاتيح التشفير).
* **`cors`**: لتأمين والسماح بالاتصال بالسيرفر من نطاقات مختلفة (لوحة الأدمن وتطبيق الفرونتيند).
* **`bcryptjs`**: لتشفير وهاش وتأمين كلمات مرور المستخدمين قبل حفظها في قاعدة البيانات.
* **`jsonwebtoken` (JWT)**: لإنشاء وتوثيق الـ Tokens الخاصة بصلاحيات تسجيل الدخول وحماية المسارات.
* **`socket.io`**: لبناء سيرفر الـ WebSockets لإدارة غرف الشات اللحظية بين البائع والمشتري.
* **`node-cron`**: لتشغيل مهام مؤتمتة وتلقائية في خلفية السيرفر (مثل فحص عروض RFQ منتهية الصلاحية يومياً).
